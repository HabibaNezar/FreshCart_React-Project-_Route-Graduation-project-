# FreshCart E-commerce Project

FreshCart is a modern, fully functional e-commerce application built with React. It provides a seamless shopping experience with features like user authentication, product browsing, cart management, wishlist functionality, and a secure checkout process.

## 🚀 Live Demo
[View Live Demo](https://HabibaNezar.github.io/E-commerce-freshcart)

## ✨ Features

- **User Authentication:** Secure registration, login, and password recovery functionality.
- **Product Management:** Browse products by categories and brands with real-time data fetching.
- **Shopping Cart:** Add, remove, and update product quantities in the cart.
- **Wishlist:** Save favorite products for later viewing.
- **Checkout Process:** Integrated checkout flow for online payments.
- **Responsive Design:** Optimized for all devices using Tailwind CSS and Flowbite.
- **Dynamic UI:** Smooth transitions and sliders using React Slick.
- **State Management:** Efficient data fetching and caching with React Query.

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS, Flowbite, CSS Modules
- **Routing:** React Router Dom
- **Data Fetching:** Axios, TanStack React Query
- **Form Handling:** Formik, Yup (Validation)
- **Icons & UI:** FontAwesome, React Slick (Carousel)
- **Notifications:** React Hot Toast

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HabibaNezar/FreshCart_React-Project-_Route-Graduation-project-.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd app
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure

```text
src/
├── Components/         # Reusable UI components
│   ├── Authentication/ # Login, Register, ForgetPassword
│   ├── Brands/         # Brand-related components
│   ├── Cart/           # Cart management
│   ├── Categories/     # Category browsing
│   ├── Checkout/       # Checkout flow
│   ├── Home/           # Landing page & sliders
│   └── WishList/       # Wishlist functionality
├── Context/            # React Context for global state (Cart, User, Wishlist)
├── Hooks/              # Custom React Hooks
└── assets/             # Images and static files
```
